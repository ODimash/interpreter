# Грамматика языка
### Программа
```
<программа> ::= <инструкция> | <программа> <инструкция>
```

### Инструкции
```

<инструкция> ::= <оператор> | <оператор> <разделитель>
                | <условная_конструкция> | <цикл> | <функция> | <вызов_функции>
```

### Операторы
```
<оператор> ::= <присваивание> | <выражение> | <вывод_сообщения> | <выполнение_команды>
             | <встроенная_функция>
```

### Переменные и присваивание
```
<переменная> ::= <идентификатор>

<присваивание> ::= <переменная> ":=" <выражение>
```

### Выражения
```
<выражение> ::= <число> | <строка> | <булево_значение> | <переменная>
              | <выражение> <оператор_сравнения> <выражение> | "(" <выражение> ")"
```

### Условные конструкции
```
<условная_конструкция> ::= "if" "(" <выражение> ")" "{" <программа> "}"
                         | "if" "(" <выражение> ")" "{" <программа> "}" "else" "{" <программа> "}"
```

### Циклы
```
<цикл> ::= "Loop" "(" <число> ")" "{" <программа> "}"
```

### Функции
```
<функция> ::= "Function" <идентификатор> "(" ")" "{" <программа> "}"
```

### Вызов функции
```
<вызов_функции> ::= <идентификатор> "(" ")"
```

### Вывод сообщения
```
<вывод_сообщения> ::= "MsgBox" "(" <строка> ")"
```

### Ввод сообщения
```
<ввод_сообщения> ::= "InputBox" "(" <строка> ")"
```

### Встроенные функции
```
<встроенная_функция> ::= <вывод_сообщения> | <выполнение_команды>
```

### Терминальные символы
```
<идентификатор> ::= <буква> | <идентификатор> <буква> | <идентификатор> <цифра>
<строка> ::= <кавычки> <символы> <кавычки>
<число> ::= <цифра> | <число> <цифра>
<буква> ::= "a" | "b" | ... | "z" | "A" | "B" | ... | "Z"
<цифра> ::= "0" | "1" | ... | "9"
<кавычки> ::= "\""
<оператор_сравнения> ::= "=" | "<>" | "<" | ">" | "<=" | ">="
<разделитель> ::= ";" | ","
```


# Пример кода


```
# Определение функции
Function Greet() {
    MsgBox("Hello, world!")
}

# Вызов функции
Greet()

# Цикл для вывода чисел от 1 до 5
Loop(5) {
    MsgBox(i)
}

# Условная конструкция
if (i > 3) {
    MsgBox("i is greater than 3")
} else if (i = 3) {
    MsgBox("i is less than or equal to 3")
}

# Присваивание переменной
x := 10

InputBox(x, "Загаловок", "Подсказка")

# Вывод значения переменной
MsgBox(x, "Загаловок")
```

Просто вставьте ваш код внутри блока кода и укажите язык программирования после первых трех обратных апострофов.
